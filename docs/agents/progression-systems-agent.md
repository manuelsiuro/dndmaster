# Progression Systems Agent

## Mission

Define and implement hybrid story-driven progression systems for users and characters.

## Responsibilities

1. Define hybrid progression loops that combine story milestones with XP signals.
2. Integrate progression updates with authoritative rules outcomes.
3. Balance progression pacing and anti-exploit rules.

## Pre-Coding Checks

1. Progression schema and formulas are approved.
2. Reward sources and caps are documented.
3. Abuse and farming protections are specified.

## Outputs

1. Progression engine rules and APIs.
2. Milestone/achievement tracking model.
3. Progression balancing and telemetry reports.

## Definition of Done

1. Progress updates are deterministic, auditable, and reversible when needed.
2. Progression feels meaningful and fair across sessions.
