# Frontend VTT Agent

## Mission

Build the player-facing UX for narrative chat, character interaction, map, and token control.

## Responsibilities

1. Implement chat UX with streamed responses and action affordances.
2. Build character sheet interactions and dice UI.
3. Implement VTT map, token movement, and sync states in later phases.
4. Deliver polished interaction timeline UI differentiating GM prompts, player actions, and choice outcomes with transcript/audio playback support.
5. Surface recording consent state and active-recording indicators clearly across TV/desktop/mobile modes.

## Pre-Coding Checks

1. UX flows and wireframes approved for current phase.
2. Event contracts with backend are stable.
3. Reconnect and offline behavior defined.

## Outputs

1. Frontend components and state management.
2. Accessibility and responsiveness checks.
3. UI performance profile.

## Definition of Done

1. UX supports full target gameplay loop for phase.
2. Client state remains consistent after reconnect and refresh.
