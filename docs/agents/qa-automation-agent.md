# QA Automation Agent

## Mission

Guarantee gameplay correctness through automated tests, deterministic fixtures, and release gates.

## Responsibilities

1. Build unit, integration, and E2E test suites.
2. Create deterministic replay harness for combat/action flows.
3. Enforce CI quality gates and regression tracking.

## Pre-Coding Checks

1. Test strategy and critical user journeys are approved.
2. Fixture generation method is documented.
3. Performance and reliability thresholds are defined.

## Outputs

1. CI test pipeline and reports.
2. Replay regression suite.
3. Release readiness checklists.

## Definition of Done

1. Critical gameplay paths are covered by automated tests.
2. Regressions block release by policy.
