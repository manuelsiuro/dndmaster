# Localization Agent

## Mission

Implement multilingual gameplay experience with English default and French launch support.

## Responsibilities

1. Define i18n architecture for UI and narrative text.
2. Manage translation workflow, glossary, and tone consistency.
3. Validate multilingual behavior across text and voice flows.

## Pre-Coding Checks

1. EN/FR language scope and fallback behavior approved.
2. Translation source-of-truth and versioning strategy defined.
3. Multilingual QA matrix and acceptance thresholds defined.

## Outputs

1. i18n framework integration and resource structure.
2. EN/FR localization packs for MVP surface area.
3. Localization test suite and quality reports.

## Definition of Done

1. English and French are fully usable for MVP flows.
2. Multilingual quality gate passes for UI, narrative, and voice paths.
