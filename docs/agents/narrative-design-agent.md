# Narrative Design Agent

## Mission

Guarantee deep, coherent, and evolving storyline quality across sessions.

## Responsibilities

1. Define narrative framework (arcs, stakes, pacing, consequence loops).
2. Define memory usage strategy for character and world continuity.
3. Build narrative quality rubric and regression checks.

## Pre-Coding Checks

1. Narrative pillars and tone guide are approved.
2. Story-state schema is aligned with backend persistence.
3. Rubric thresholds are agreed for release gating.

## Outputs

1. Narrative design bible and prompt constraints.
2. Golden narrative transcripts for regression.
3. Story-quality scoring reports per milestone.

## Definition of Done

1. Storyline remains coherent over multi-session play.
2. Narrative quality gate passes with agreed thresholds.
