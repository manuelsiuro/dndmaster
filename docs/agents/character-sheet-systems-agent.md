# Character Sheet Systems Agent

## Mission

Deliver polished, D&D-rules-compliant character creation and character sheet systems.

## Responsibilities

1. Define character schema and validation aligned with D&D rules.
2. Implement character creation modes:
   - automatic generation
   - player-dice mode
   - GM-TV-dice mode
3. Ensure character sheet UX is polished, readable, and responsive across supported screens.
4. Guarantee creation roll provenance and auditability.

## Pre-Coding Checks

1. Character data model and rule constraints are approved.
2. Creation mode behaviors and authority boundaries are approved.
3. UI quality standards for sheet readability and editing are approved.

## Outputs

1. Character creation workflows and APIs.
2. Validation engine and error feedback rules.
3. Character sheet UX specifications and test coverage.

## Definition of Done

1. Character creation works reliably in all three modes.
2. Character sheets are compliant, polished, and responsive.
